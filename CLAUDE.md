# Orçamento Brum — Proposta Comercial

## Contexto
Proposta comercial (single-page HTML) para o cliente Brum Chopp.
MVP de plataforma de delivery de chopp com automação e agenda.
Briefing completo do cliente em `BRIEFING.md`.

## Stack
- HTML estático com Tailwind CSS (CDN via build local)
- Fonte: Inter (Google Fonts)
- Deploy: Vercel

## Estrutura
- `index.html` — página única da proposta
- `src/input.css` → `dist/output.css` — Tailwind build
- `proposta-full.png` — screenshot de referência do design

## Decisões de projeto
- Prazo de entrega: **40 dias** (não usar 45 dias)
- Valor: R$ 15.000 dividido em 3x de R$ 5.000
- Parcelas: aprovação, MVP base (~dia 20), entrega final (dia 40)
- Timeline: 6 semanas (Sem. 1–2, 3–4, 5, 6)
- Abordagem: código próprio (não no-code) — argumento central da proposta
- Tom: direto, profissional, sem exageros — fala de igual pra igual com o cliente

## Seções da proposta
1. Hero
2. O que você pediu (problema/contexto)
3. Como o sistema vai funcionar (fluxo visual)
4. O que você terá (escopo — 6 módulos)
5. Timeline semana a semana
6. Investimento (R$ 15.000)
7. Como funciona o pagamento (3 parcelas)
8. Diferenciais
9. Sobre mim
10. CTA final
11. Footer
