# Proposta Comercial — MVP Delivery de Chopp

## What This Is

Página HTML estática de proposta comercial/orçamento para apresentar ao cliente um MVP de plataforma de delivery e locação de chopp para eventos. A proposta será hospedada na Vercel e enviada como link profissional. Visual clean corporativo, valor fechado de R$ 10.000, prazo de 45 dias.

## Core Value

A proposta deve transmitir profissionalismo e competência técnica, detalhando o escopo completo do MVP de forma clara para que o cliente aprove o investimento.

## Context

O cliente publicou uma vaga buscando especialista em automação/no-code para um MVP de delivery de chopp. O escopo do MVP inclui:

1. **Landing page** — Catálogo de produtos (marcas, litros, chopeiras, extras), diferenciais, CTAs
2. **Formulário de pedidos** — Campos personalizados (data, endereço, quantidade, tipo chopeira, acessórios), integração CRM
3. **Automação WhatsApp** — Confirmação de pedidos, lembretes, solicitação de documentos, pós-venda
4. **Pagamentos** — Pix e link de pagamento (cartão), confirmação automática após pagamento
5. **Painel interno** — Controle de pedidos, status (novo→confirmado→em rota→entregue→recolhido→finalizado), filtros por data/região
6. **Estrutura escalável** — Preparada para expansão futura a marketplace com parceiros distribuidores

O cliente mencionou preferência por no-code/low-code mas não é obrigatório. Marcus tem mais experiência com código (Next.js, Node) e vai propor stack code com auxílio de AI.

Prazo desejado pelo cliente: 30-45 dias.
Forma de contratação: projeto com possibilidade de continuidade.

## Constraints

- **Tech stack**: HTML/CSS/JS estático — single page, sem framework pesado
- **Hospedagem**: Vercel (gratuito para sites estáticos)
- **Visual**: Clean corporativo — fundo branco, tipografia limpa, profissional e neutro
- **Idioma**: Português brasileiro
- **Responsividade**: Mobile-first, funcionar bem em celular (cliente pode abrir no WhatsApp)
