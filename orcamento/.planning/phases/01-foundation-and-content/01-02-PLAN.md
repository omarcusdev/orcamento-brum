## Goal

Write the complete semantic HTML skeleton for all 11 sections of the proposal page in the correct AIDA funnel order. - `index.html` contains exactly 11 sections in this order: `#hero`, `#problema`, `#solucao`, `#escopo`, `#timeline`, `#investimento`, `#pagamento`, `#diferenciais`, `#sobre`, `#cta`, then `<footer>`

- All six scope modules appear inside `#escopo` as list items with an `<h3>` heading each
- All section `<h2>` headings are result-oriented — none use labels like "Escopo do trabalho" or "Sobre mim"

- HTML validates without errors (no unclosed tags, no duplicate IDs, no missing alt attributes on any `<img>`)
